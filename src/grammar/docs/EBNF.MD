# Simple Guide to EBNF

EBNF stands for **Extended Backus–Naur Form**. It is a notation used to describe the syntax of programming languages among other things.

## Key
| Symbol | Meaning |
|---------|---------|
| `=` or `::=` | defines a rule |
| `\|` | or |
| `,` or `space` | sequence |
| `;` or `.` | end of rule |
| `[ ... ]` or `?` | zere or one time |
| `{ ... }` or `*` | zero or more times |
| `( ... )` | grouping |

## Terminals vs Non-Terminals

Terminals should be denoted in **UPPERCASE** and non terminals should be written in **lowercase**.
